# Site TOMIX

Site vitrine statique (HTML/CSS/JS) pour la marque TOMIX — Saveurs du Terroir.

## Structure
```
tomix-site/
├── index.html        (Accueil)
├── produits.html      (Fiches produits + cartons pro)
├── contact.html        (Formulaire + coordonnées)
├── css/style.css
├── js/script.js
└── images/            (vos visuels produits)
```

## Utilisation dans VS Code
1. Dézippez le dossier `tomix-site`.
2. Ouvrez le dossier dans VS Code.
3. Installez l'extension "Live Server" (ou équivalent) puis clic droit sur `index.html` → "Open with Live Server".

## À personnaliser
- Numéro WhatsApp / téléphone : recherchez `221771234567` dans les fichiers HTML et remplacez par votre vrai numéro.
- Email : `contact@tomix.sn` dans `contact.html`.
- Le formulaire de contact est actuellement une simulation côté client (pas de backend). Pour recevoir les messages, branchez-le sur un service comme Formspree, EmailJS, ou votre propre backend.
- Réseaux sociaux : liens `#` dans le footer, à remplacer par vos vraies pages.
